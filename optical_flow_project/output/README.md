# Output Directory

This directory stores all output generated by the optical flow processing pipeline.

## Output Structure

When you run the pipeline, the following outputs may be generated:

- **Extracted Frames**: `frames/` subdirectory containing individual video frames
- **Annotated Videos**: Videos with object detection bounding boxes
- **Optical Flow Videos**: Videos showing trajectory predictions and motion analysis
- **Results JSON**: `pipeline_results.json` containing processing statistics

## Example

```
output/
├── frames/
│   ├── frame_0001.png
│   ├── frame_0002.png
│   └── ...
├── annotated_video.mp4
├── optical_flow_prediction.mp4
└── pipeline_results.json
```

## Cleanup

You can safely delete the contents of this directory to free up space. The pipeline will recreate it as needed.
