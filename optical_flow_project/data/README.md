# Data Directory

This directory is designated for storing input video files and related data.

## Usage

Place your video files in this directory before processing. Supported formats include:
- `.mp4`
- `.avi`
- `.mov`
- `.mkv`

## Example

```
data/
├── video1.mp4
├── video2.mp4
└── sample_traffic.avi
```

You can also organize videos in subdirectories for better organization.
